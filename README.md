# Single-spa React MFE w/ Rollup + esbuild

Just a proof of concept

## Features

- Rollup + Esbuild for blazing fast reload times
- Did I just say that is pretty fast?
- Css Loading

## Run Locally

Install deps with your favorite dependecy manager

```bash
  pnpm install
```

Run locally

```bash
  pnpm run dev
```

Run build

```bash
  pnpm run build
```
